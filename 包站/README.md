# 包站 - 视频分享平台

一个仿照B站（哔哩哔哩）风格设计的视频分享网站。

## 功能特性

### 🎯 核心功能
- **视频展示**: 网格布局展示视频列表
- **视频播放**: 点击视频卡片打开播放模态框
- **搜索功能**: 支持按标题、作者、分类搜索
- **分类筛选**: 按游戏、动画、音乐等分类筛选视频
- **排行榜**: 按播放量排序的热门视频排行榜
- **轮播图**: 自动轮播的首页横幅

### 🎨 界面设计
- **响应式设计**: 支持桌面端和移动端
- **B站风格**: 采用B站经典的蓝色主题配色
- **现代化UI**: 圆角设计、阴影效果、平滑动画
- **交互体验**: 悬停效果、点击反馈、加载动画

### 📱 用户体验
- **流畅动画**: 页面滚动时的渐入动画
- **键盘支持**: ESC键关闭视频播放
- **自动播放**: 轮播图自动切换
- **加载优化**: 图片懒加载和性能优化

## 使用方法

### 1. 直接打开
双击 `index.html` 文件即可在浏览器中打开网站。

### 2. 本地服务器（推荐）
为了获得最佳体验，建议使用本地服务器：

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx http-server

# 然后在浏览器中访问
http://localhost:8000
```

## 功能说明

### 视频播放
- 点击任意视频卡片打开播放模态框
- 支持视频控制（播放/暂停、音量、全屏等）
- 点击关闭按钮或按ESC键关闭播放器

### 搜索功能
- 在顶部搜索框输入关键词
- 支持搜索视频标题、作者、分类
- 实时显示搜索结果数量

### 分类筛选
- 点击首页的分类图标（游戏、动画、音乐等）
- 自动筛选对应分类的视频
- 显示分类下的视频数量

### 排行榜
- 按播放量自动排序
- 前三名有特殊样式标识
- 点击排行榜项目可直接播放视频

## 自定义配置

### 添加新视频
在 `script.js` 文件中的 `videoData` 数组中添加新的视频对象：

```javascript
{
    id: 9,
    title: "视频标题",
    thumbnail: "缩略图URL",
    duration: "时长",
    views: "播放量",
    author: "作者",
    authorAvatar: "作者头像URL",
    date: "发布时间",
    category: "分类",
    videoUrl: "视频文件URL",
    description: "视频描述"
}
```

### 修改样式
- 主色调：修改CSS中的 `#00a1d6` 颜色值
- 布局：调整 `style.css` 中的网格和间距设置
- 动画：修改CSS动画参数和JavaScript动画逻辑

### 添加新分类
1. 在HTML中添加分类图标
2. 在CSS中设置分类样式
3. 在JavaScript中添加分类筛选逻辑
